1. 项目概述

PDF工具专业版是一款功能强大的桌面PDF处理工具，集成了文档查看、文本编辑、OCR识别和批量处理等核心功能。

本项目采用现代化的技术架构，基于Electron + React + TypeScript构建，提供跨平台的桌面应用体验。

2. 核心功能

智能PDF文本编辑
高精度OCR文字识别
批量处理自动化
响应式用户界面

3. 技术架构

前端框架: React 18 + TypeScript
桌面框架: Electron 27
状态管理: Zustand
构建工具: Vite
